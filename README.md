# break08s-modpack
This project include: Super Optimization Pack, Total Resource Pack and MoreEz modpacks on Modrinth (now are all under review)

Modpacks made with [PACKWIZ](https://github.com/packwiz/packwiz)

# Note

- If you use packwiz to export to .zip file for modpacks, don't use `packwiz curseforge export`, it won't export all mods in mod pack, export modpacks to .mrpack and use [MRPACK to ZIP Converter](https://mrpacktozip.com/) instead of to convert mrpack to zip.
